# DTX SDR Leaders Netlify Project

This is a Vite + React project prepared from your landing page preview.

## Run locally

```bash
npm install
npm run dev
```

## Build for production

```bash
npm install
npm run build
```

The production build will be created in `dist/`.

## Deploy to Netlify

### Option 1: GitHub + Netlify
1. Upload this folder to a new GitHub repository.
2. In Netlify, choose **Add new project** -> **Import an existing project**.
3. Select your GitHub repo.
4. Netlify should use:
   - Build command: `npm run build`
   - Publish directory: `dist`
5. Set the site name to `dtxsdrleaderwaitlist` if available.

### Option 2: Manual deploy
1. Run `npm install`
2. Run `npm run build`
3. Upload the `dist/` folder to Netlify.

## Domain setup
- Free Netlify URL: `dtxsdrleaderwaitlist.netlify.app` if that site name is available.
- Custom domain example: `dtxsdrleaderwaitlist.com`
